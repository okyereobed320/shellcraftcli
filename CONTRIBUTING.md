# 🤝 Contributing to Shellcraft

First, thank you for considering contributing to Shellcraft! It's people like you that make it a great tool.

## 📋 Ways to Contribute

1.  **Add Questions:** Add more questions to the `data/linux.json` or create new modules (e.g., `docker.json`).
2.  **Improve the CLI:** Add new commands or improve the current experience.
3.  **Report Bugs:** Found something wrong? Open an issue!
4.  **Documentation:** Help improve the README or this guide.

## 🛠️ Development Setup

```bash
# Fork and clone the repo
git clone https://github.com/your-username/shellcraft.git

# Install dependencies
npm install

# Run the CLI in development mode
node bin/shellcraft.js start linux
```

## 📝 Commit Messages

We prefer clear, concise commit messages:
- `feat: add docker module questions`
- `fix: correct typo in linux quiz`
- `docs: update roadmap`
