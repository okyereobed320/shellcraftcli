# 🚀 Shellcraft

**Master the terminal. Craft your engineering skills.**

Shellcraft is an open-source, CLI-based training platform designed to help you master Linux, Cloud, and DevOps through hands-on practice directly in your terminal.

---

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/okyereobed320/shellcraftcli.git

# Navigate to the directory
cd shellcraft

# Install dependencies
npm install

# Link the command (optional, for local development)
npm link
```

## 🎯 Usage

```bash
# List available modules
shellcraft list

# Start the Linux module
shellcraft start linux

# View your progress
shellcraft score
```
